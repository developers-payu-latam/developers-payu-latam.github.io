PROCESSAMENTO DE TRANSAÇÕES
 
Contamos com diversos métodos para o processamento de transações:

Todas as operações disponíveis são:

  • Autorização: usada para verificar se um cartão de crédito se encontra ativa, se possui crédito, etc. A transação não é finalizada até que seja enviada uma transação de captura (somente disponível para contas do Brasil).
  • Captura: finaliza uma transação previamente autorizada. É aqui quando se realiza o débito da conta do cartão (somente disponível para contas do Brasil).
  • Autorização e captura: este é o tipo de transação mais utilizada. Com esta opção é enviada a autorização o montante da transação e se for aprovada realiza-se imediatamente a captura.
  • Anulação:: Usada para reverter uma transação previamente autorizada (somente disponível para contas do Brasil).
  • Reembolso: Usada para reverter uma transação previamente capturada.
